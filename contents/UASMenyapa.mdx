##### Introduction

UAS Menyapa is a dynamic event ticketing web application designed to simplify the ticket purchasing process. With an intuitive interface and smart functionalities, it enables users to quickly buy tickets and access events via secure QR code validation.

##### Project Details

This full-featured ticketing platform empowers both event organizers and attendees. Visit the official website at [UAS Menyapa](https://uasmenyapa.com) for more information.

##### Key Features

- **Seamless Ticket Purchases:** Enjoy an effortless and fast checkout process.
- **Instant QR Code Generation:** Each ticket automatically generates a scannable QR code.
- **Efficient Ticket Scanning:** Optimized scanning functionality ensures smooth event access.

##### Technologies Used

- **Framework:** Next.js
- **Programming Language:** TypeScript
- **State Management:** Jotai
- **Database:** mySQL
- **ORM:** Drizzle ORM
- **Authentication:** Auth.js V5
- **UI Framework:** Shadcn UI
- **CSS Framework:** Tailwind CSS
